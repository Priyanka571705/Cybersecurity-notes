# Burp Suite Basics

## What is Burp Suite?
Burp Suite is a web security testing tool
used to analyze and understand HTTP/HTTPS traffic
between a browser and a web application.

## Why Burp Suite is Used
- To intercept web requests and responses
- To understand how web applications communicate
- To analyze cookies, headers, and parameters

## Basic Components
- Proxy
- Repeater
- Intruder
- Target

## Simple Use Case
Burp Suite allows users to see
what data is sent from the browser to the server.

## Important Note
Burp Suite should be used only
on applications you own or have permission to test.
