# HTTP vs HTTPS

## What is HTTP?
HTTP (HyperText Transfer Protocol)
is used for communication between
a browser and a server.

## What is HTTPS?
HTTPS is the secure version of HTTP.
It encrypts data using SSL/TLS.

## Key Differences

| HTTP | HTTPS |
|----|------|
| Data is not encrypted | Data is encrypted |
| Less secure | More secure |
| Port 80 | Port 443 |

## Simple Example
HTTP data can be read by attackers,
HTTPS protects sensitive data.

## Simple Note
HTTPS is recommended for secure communication.
