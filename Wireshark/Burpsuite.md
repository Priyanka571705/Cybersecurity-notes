# Burp Suite

## Initial Understanding
Before Burp Suite, web communication felt abstract.

## After Using Burp Suite
I could clearly see:
- Request methods (GET, POST)
- Headers and cookies
- Server responses

## SOC Perspective
Understanding web traffic is important
for identifying abnormal or suspicious behavior.

## Key Takeaway
Burp Suite improves visibility into web communication,
which is essential for security analysis.
