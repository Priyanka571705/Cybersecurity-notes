# Wireshark Basics

## What is Wireshark?
Wireshark is a network packet analyzer
used to capture and inspect network traffic.

## Why Wireshark is Used
- To analyze network communication
- To troubleshoot network issues
- To understand protocols

## Common Protocols Seen
- HTTP
- DNS
- TCP
- UDP

## Simple Filters
 -http
 -dns
 -tcp
