# Nmap Basics

## What is Nmap?
Nmap (Network Mapper) is a network scanning tool
used to discover hosts, open ports, and running services
on a network.

## Why Nmap is Used
- To find active devices on a network
- To identify open ports
- To understand which services are running

## Basic Nmap Commands
nmap <IP Address>
nmap -sV <IP Address>
nmap -p- <IP Address>
nmap -A <IP Address>


### Scan a single target
nmap 192.168.1.1
nmap example.com
